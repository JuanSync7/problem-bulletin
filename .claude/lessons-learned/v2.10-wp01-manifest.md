# v2.10-WP01 Manifest — v1 test sunset

Audit trail of all 24 files that contributed to the **313-failure v1 baseline**
recorded at the start of v2.10-WP01. Each file is classified and accompanied
by a one-line justification.

The chosen mechanism: a **conftest `pytest_collection_modifyitems` hook** in
`tests/conftest.py` consumes the frozenset `V1_DEFERRED_NODE_IDS` from
`tests/_v1_deferred.py` and marks every listed test ID with `pytest.mark.skip`.
No production files (`app/`) were touched. No test files were deleted —
preserving every legacy test means follow-up WPs can port them back to green
without re-discovering intent.

Classification legend:
- **ported (deferred-skip)** — file kept on disk, failing tests skipped via the
  centralised deselect manifest, with the passing tests in the same file still
  running normally.
- **delete** — file removed wholesale (only used if there were zero salvageable
  tests AND no v2 analog).
- **skipped (reason)** — entire module skipped at the module level.
- **still-failing (reason)** — file is still red after WP01.
